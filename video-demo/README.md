# GovBridge GTM Video Demo Prototype

Prototipo Remotion para el video demo de hackaton de 3 minutos.

## Que incluye

- Timeline de 180 segundos en `src/DemoVideo.tsx`.
- Storyboard y guion en `storyboard.md`.
- Prompts para HeyGen/ElevenLabs en `production-prompts.md`.
- Placeholders para:
  - Avatar de Andrea.
  - Grabacion de Claude Code.
  - Notion llenandose.
  - Capturas de entregables comerciales.

## Instalar y correr

Desde esta carpeta:

```bash
npm install
npm run preview
```

Para renderizar:

```bash
npm run render
```

## Assets finales

Copia los archivos reales en `public/media/` con estos nombres sugeridos:

- `andrea-hook.mp4`
- `claude-prompt.mp4`
- `claude-code-run.mp4`
- `notion-fill.mp4`
- `notion-deliverables.mp4`
- `andrea-close.mp4`

La version actual usa placeholders visuales, asi que puede revisarse antes de tener los videos reales.
